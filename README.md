1. Inventory

Affiche les informations système :

Système d’exploitation
Version du noyau
Processeur
Mémoire RAM
Espace disque
Adresse IP

2. Backup

Permet de sauvegarder un dossier sous forme d’archive.

3. Cleanup

Supprime les fichiers inutiles (.tmp, .log).

4. Report

Génère un rapport contenant :

Informations système
Sauvegardes réalisées
Résumé des actions
